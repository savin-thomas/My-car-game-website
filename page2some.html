The Graphics

The 2 main things

-The Pygame module

Pygame module is a module in Python which deals with graphics and is used to make games.
There are several functions within this module which will be very helpfull to draw objects on screen such as rectangles, lines, images, text, etc.

-The colors to alpha property

The colors to alpha property is a property where you can change a particular color of an image to transparent so that the image blends with the background.

Now let's go step by step :-

Step 1: Draw the road with it's yellow lines and white dividers :-

Note that the entire window screen is in the form of a cooridinate system, with x axis incrementing from left to right and y axis incrementing from top to botttom and each unit is a pixel.
After determining the window screen's width and height (in my case it is 1300 pixels wide and 700 pixels in height), we draw a grey rectangle for the road using the pygame.draw.rect function.
Then we will draw 4 yellow lines and 46 dividers using the pygame.draw.line function.

Step 2: Draw the cars and the road objects like tins and board :-

We first download the images of cars and the road objects like tins and board and then we crop them, find out their dimensions which will be useful in coding, edit the background color of the images using the colors to alpha property, where we select a particular color to become transparent.
This can be done with some colors to alpha websites online where we can upload our image, select the color we want to turn transparent and save the image.
Now our cars, tins and board will blend with the road behined it.
I had to also make use of 'paint' to edit the images.
After editing the images, store them in the python directory as .png images.
Then, we can draw them on the window screen using screen.blit function which will be done later in the coding part.

Step 3: Draw the score and the 'You Crashed message'.

Here we use the screen.blit function again to draw text like 'Score:' and 'You Crashed' on the window screen with a particular font style and size (in my case I used 'freesansbold.ttf' as font style and 36, 60 as respective font sizes).
